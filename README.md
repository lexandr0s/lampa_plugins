# lampa_plugins

- **filter_mod.js** - улучшение к стандартному фильтру Лампы. Добавляется отдельным пунктом в главное меню как "Фильтр+". Сохраняет настройки фильтра между сессиями, позволяет одной кнопкой сбросить настройки языков и жанров. По функционалу полностью идентичен штатному фильтру Лампы.
